                                                                        Hi Joel Peyton, thanks for the kind words about our work! I'm glad we were able to help you with leveling up your skills. To answer your questions:

                                                                        1. $stmt is just my personal preference. It is short for "SQL statement". You can change it to your liking.

                                                                        2. Using bindParam allows us to use prepared statements. It is a feature used to execute similar SQL statements repeatedly with high efficiency. It minimize bandwidth to the server as you need send only the parameters each time, and not the whole query
